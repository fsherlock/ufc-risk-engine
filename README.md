# UFC Risk Engine Documentation

### Overview
The UFC Risk Engine is designed to analyze and compute risks related to UFC (Ultimate Fighting Championship) events.

### Features
- **Risk Analysis**: Calculate potential risks based on fighter statistics.
- **Event Simulation**: Simulate fights based on historical data.
- **Statistical Insights**: Provide insights through historical analysis and predictive modeling.

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/fsherlock/ufc-risk-engine.git
   ```
2. Change into the project directory:
   ```bash
   cd ufc-risk-engine
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Usage
To use the UFC Risk Engine, execute the following command:
```bash
node index.js
```

# UFC MarX Risk Engine

A powerful risk analysis and prediction engine for UFC fights.

## Features
- **Risk Engine**: Calculate optimal betting strategies (Kelly, Equal Stake, YOLO).
- **Tale of the Tape**: Visual comparison of fighter stats.
- **Dynamic Search**: Auto-complete search for hundreds of UFC fighters.
- **PWA Ready**: Installable as a mobile app.

## How to Use
1. Enter the number of fights and your bankroll.
2. Search and select Fighter A and Fighter B.
3. The engine auto-fills stats (Reach, Height, Stance).
4. Adjust confidence levels and view the "Tale of the Tape".
5. Click **Calculate Strategies** to see the risk analysis.

## Installation (Mobile)
1. Open the hosted link on Chrome (Android) or Safari (iOS).
2. Tap "Add to Home Screen".
